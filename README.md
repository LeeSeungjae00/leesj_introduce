# 구상
* 자기소개 포트폴리오 형식으로 만들예정
* 각 게임을 클리어 해가면서 나를 알아가는 방식으로 설계해 볼 것임
* 난이도 설정이 있어 쉽게 나를 알아갈건지 어렵게 알아갈건지 선택할 수 있도록 
* 지금 생각해놓은 게임 5가지 정도로 할 예정 일단 생각나는 것들만 구현 하고 그 후 추가하도록 하자
    - 끝말잇기(API 사용)
    - 반응속도 테스트
    - 마우스 클릭속도 테스트(speedmeter 사용하자)
* 스크롤에니메이션을 통해서 좀더 눈에띄도록 하자
* 테스트 코드도 공부할 겸 짜보자
* 나중에 Back-end 부분도 node 를 통해 각 난이도 별로 클리어한 사람, 점수 기록 등등을 DB까지 연동하여 만들어 배포까지 해보자
